Development of a C++ compiler in Python  
  
In the Compilation.py file we import enum and sys.  
To write the source code in c, we write in the test.c file and the compiler takes care of compiling this test.c file  
The result (the compiled code) is then sent to the code_compile.txt file  
---  
Dans le fichier Compilation.py, nous importons enum et sys.  
Pour écrire le code source en c , nous écrivons dans le fichier test.c et le compilateur s'occupe de compiler ce fichier test.c  
Le résultat (le code compilé) est ensuite envoyé sur le fichier code_compile.txt
